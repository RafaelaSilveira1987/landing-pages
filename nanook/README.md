# Nanook Fitness Landing Page

Landing page moderna, minimalista e responsiva para vender o Nanook Fitness App para academias.

## Arquivos

- `index.html`: estrutura da landing page
- `assets/css/style.css`: estilos, responsividade e visual
- `assets/js/app.js`: menu mobile, animações e botão voltar ao topo com progresso

## Ajustes rápidos

1. Troque o link do WhatsApp no botão principal:
   - procure por `https://wa.me/5500000000000`
   - substitua pelo número oficial com DDI e DDD.

2. Caso tenha a logo oficial:
   - substitua o bloco `.brand-mark` por uma imagem, ou me envie a logo que eu encaixo no topo.

3. Para subir no servidor:
   - envie a pasta inteira para o diretório público desejado.
